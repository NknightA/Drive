# Drive

## What's this?

this is Google Drive client.

## How to USE?

1.
```
git clone https://github.com/NknightA/Drive.git

```

2.

```
npm i
```

3-1 test

```
npx electron .
```

3-2 build 

```
npx electron-packager . drive
```

for Windows
```
npx electron-packager . drive --win32 --x64
```

Out drive dir.
